# Dataset

This directory contains documentation related to the dataset used in this project.

The actual image files are not included in the repository due to size and licensing constraints.

## Expected Structure

After downloading the dataset, the `data/` directory should have the following structure:

```
data/
├── img_train/
├── img_test/
├── train_answers.csv
└── activity_categories.csv
```

## Notes

- The dataset is used only within the scope of this project.
- No external data sources were used.
- Please ensure that the directory structure matches the one above before running the code.
